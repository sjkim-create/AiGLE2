## 1. 화면 개요

- **화면 ID**: `SET-01`
- **화면 이름**: 환경설정 (Settings)
- **대상 사용자**: 로그인된 사용자 (AiGLE 서비스 이용자)
- **비즈니스 목적**: 사용자가 채점 전 환경설정을 셋팅할 수 있도록 하여 채점이 지연되는 것을 방지한다. 연결된 펜 디바이스의 펌웨어 및 소프트웨어를 직접 관리하고, 기기 초기화 등 주요 디바이스 운영 작업을 수행할 수 있도록 지원한다.

## 2. 주요 사용자 흐름 (Happy Path)

1. 사용자가 왼쪽 사이드바 상단 프로필 아이콘을 클릭하여 드롭다운 메뉴를 열고 [환경설정]을 선택한다.
2. 시스템이 설정 전용 사이드바(내 정보 / 고객센터 / 환경설정)로 전환하고, 환경설정 메인 화면을 카드형 레이아웃으로 표시한다.
3. 사용자가 원하는 기능 카드(AiGLE Connect 관리 / 펜 펌웨어 업데이트 / 기기 초기화)에서 액션을 수행한다.
4. 각 액션에 따라 다운로드·업데이트·초기화가 처리되고, 결과가 화면 또는 토스트 알림으로 피드백된다.

## 3. 화면 구성

### AiGLE Connect 관리 카드

**전체 공통:**

- 현재 버전 및 최신 버전 정보 표시 (설치 여부에 따라 "설치되지 않음" 또는 버전 문자열)
- [최신 버전 다운로드] 버튼 클릭하여 **AiGLE Connect 최신 버전 설치 파일 다운로드**
  - 성공: "AiGLE Connect 다운로드가 시작되었습니다." 토스트 메시지 노출 및 다운로드 프로세스 시작
  - 완료: 다운로드 완료 시 토스트 메시지 자동 소멸 (또는 성공 안내로 교체)
  - 조건: 소프트웨어 업데이트가 필요한 경우에만 활성화 (버전 비교 기반)

| 상태             | 현재 버전 표시  | 최신 버전 표시    | 버튼 상태 |
| :--------------- | :-------------- | :---------------- | :-------- |
| 설치되지 않음    | "설치되지 않음" | X.X.X             | 활성화    |
| 업데이트 필요    | X.X.X           | X.X.X (상위 버전) | 활성화    |
| 최신 버전 설치됨 | X.X.X           | X.X.X (동일)      | 비활성화  |

### 펜 펌웨어 업데이트 카드

**전체 공통:**

- 연결된 펜 총 개수 표시 및 업데이트 필요 배지 노출
- [펜 목록 및 펌웨어 업데이트] 버튼 클릭하여 **펌웨어 업데이트 모달(`SET-01:펌웨어모달`) 오픈**
  - 성공: 펌웨어 업데이트 모달 노출
  - 조건: 연결된 펜이 0개이면 버튼 비활성화

### 펜 펌웨어 업데이트 모달

**전체 공통:**

- 연결된 펜 총 개수 헤더 정보 및 펜 목록 테이블 표시

| 열 항목       | 설명                                       |
| :------------ | :----------------------------------------- |
| No            | 순번                                       |
| 펜 MAC 주소   | 고유 식별자                                |
| 배터리 잔량   | % 단위 표시                                |
| 펌웨어 버전   | 현재 설치된 버전                           |
| 업데이트 상태 | `최신` / `업데이트 필요` / `업데이트 실패` |

- [일괄 업데이트 시작] 버튼 클릭하여 **전체 펜 펌웨어 업데이트 실행**
  - 성공: 업데이트 상태 `최신`으로 변경, 버튼 비활성화
  - 실패: 업데이트 실패 시 버튼 재활성화 (재시도 가능). 특정 펜 연결 끊김 시 해당 펜 항목 빨간색 `연결 실패` 표시, 나머지 펜은 계속 진행
  - 조건: '업데이트 필요' 상태인 펜이 1개 이상일 경우에만 활성화. 업데이트 진행 중에는 버튼 비활성화, 각 펜별 Progress Bar 표시. 배터리 20% 미만 펜은 "충전 필요" 경고 후 업데이트 대상 제외 _(확인 필요)_
- 펜 1개당 약 30초~1분 소요, "예상 소요 시간: 약 X분" 고지
- [닫기] 버튼 클릭하여 **모달 닫기**
  - 조건: 업데이트 진행 중 모달을 닫아도 하단 FAB(Floating Action Button)으로 진행 상태 지속 피드백

### 기기 초기화 카드

**전체 공통:**

- 위험성 안내 문구 노출
- [연결된 펜 전체 초기화] 버튼 클릭하여 **기기 초기화 확인 모달(`SET-01:초기화모달`) 오픈**
  - 성공: 초기화 확인 모달 노출
  - 조건: 연결된 펜이 0개이면 버튼 비활성화. 위험 작업 버튼 (WCAG 2.1 기준 고대비 빨간색 `#EF4444`)

### 기기 초기화 확인 모달

**전체 공통:**

- 안내 문구: "현재 N개의 펜이 연결되어 있습니다. 연결된 펜의 모든 데이터를 초기화하시겠습니까? 초기화 후 삭제된 데이터는 되돌릴 수 없습니다."
- [초기화] 버튼 클릭하여 **초기화 프로세스 실행**
  - 성공: 모달 닫힘 + "초기화가 완료되었습니다." 토스트 알림
  - 실패: "일부 기기 초기화 실패" 목록 표시, 수동 초기화 가이드 제공
  - 조건: 초기화 전 NeoStudio2Lite와 데이터 동기화 선행 처리
- [취소] 버튼 클릭하여 **모달 닫기**
- 키보드 접근성: 모달 오픈 시 Focus Trap 적용, `Tab` 키로 내부 요소 순환, `ESC` 키로 모달 닫기

## 4. 상태별 변화

### SET-01:기본

- **상태 설명**: 모든 데이터가 정상적으로 로드되고 연결된 펜이 있는 기본 상태
- **전환 트리거**: 환경설정 화면 진입 시 또는 데이터 로드 완료
- **주요 구성**:
  - 각 기능 카드(Connect 관리, 펌웨어 업데이트, 기기 초기화) 활성화
  - 펜 개수 및 버전 정보 정상 표시

### SET-01:펜없음과다운로드진행

- **상태 설명**: 현재 PC에 연결되거나 인식된 펜 디바이스가 0개인 상태
- **전환 트리거**: 펜 연결 해제 또는 초기 인식 결과 0개
- **주요 구성**:
  - [펜 펌웨어 업데이트], [기기 초기화] 카드의 실행 버튼 비활성화
  - 펌웨어 업데이트 카드 내 "연결된 펜: 0개" 표시 및 "업데이트 필요" 배지 미노출

### SET-01:API에러

- **상태 설명**: AiGLE Connect 버전 정보 또는 펌웨어 정보를 가져오는 API 통신 실패 상태
- **전환 트리거**: 서버 응답 코드 Error(500 등) 또는 타임아웃 발생
- **주요 구성**:
  - 현재 버전/최신 버전 정보 영역에 "확인 불가" 또는 "통신 오류" 메시지 노출
  - 다운로드 버튼은 오프라인 설치 지원을 위해 활성화 유지 가능 (정책에 따름)

### SET-01:펌웨어모달

- **상태 설명**: 펜 목록 확인 및 일괄 업데이트를 위한 팝업이 활성화된 상태
- **전환 트리거**: 펌웨어 업데이트 카드 내 [펜 목록 및 업데이트 확인] 버튼 클릭
- **주요 구성**:
  - 연결된 모든 펜의 상세 리스트(MAC, 배터리, 현재 버전, 업데이트 대상 여부) 노출
  - 업데이트 진행 시 실시간 Progress Bar 및 진행 상태(업데이트 중, 최신, 실패 등) 표시

### SET-01:초기화모달

- **상태 설명**: 기기 데이터 삭제 전 최종 확인을 받는 경고 팝업이 활성화된 상태
- **전환 트리거**: 기기 초기화 카드 내 [연결된 펜 전체 초기화] 버튼 클릭
- **주요 구성**:
  - 데이터 복구 불가능에 대한 강력한 경고 메시지 노출
  - 오동작 방지를 위한 [삭제] 버튼의 고대비 빨간색(#EF4444) 적용 및 최종 확인 절차 요구

### SET-01:다운로드진행

- **상태 설명**: AiGLE Connect 설치 파일 다운로드가 시작되어 진행 중인 상태
- **전환 트리거**: [최신 버전 다운로드] 버튼 클릭 시
- **메시지 정책**:
  - 시작 시: "AiGLE Connect 다운로드가 시작되었습니다." 상단 토스트 메시지 노출
  - 종료 시: 다운로드 완료(브라우저 이벤트 대기 또는 더미 타이머 종료) 시 해당 토스트 메시지 숨김 처리
- **주요 구성**:
  - 다운로드 버튼은 중복 클릭 방지를 위해 진행 중 비활성화 권장
